🚨 반드시 읽어주세요 — 데이터 백업
이 앱은 브라우저 저장(localStorage) 방식입니다.
❌ 브라우저 캐시/데이터 삭제 시 → 전부 삭제
❌ 기기 초기화/교체 시 → 전부 삭제
❌ 시크릿/프라이빗 모드 → 저장 안 됨
📘 백업/복원 사용법
1 백업하기 (주 1회 이상 권장)
1. 위의 [📥 데이터 백업] 버튼 클릭
2. JSON 파일이 자동 다운로드됨 (예: 비상아이비츠_백업_20260320.json)
3. 다운받은 파일을 안전한 곳에 보관:
✅ 구글 드라이브 — 내 드라이브에 '학원백업' 폴더 만들어서 업로드
✅ 카카오톡 나에게 보내기 — 파일 전송 (가장 간편)
✅ 네이버 MYBOX — 클라우드 업로드
✅ USB / 외장하드 — 물리적 백업
2 복원하기 (데이터 날아갔을 때)
1. 백업해둔 JSON 파일을 기기에 다운로드
2. 위의 [📤 데이터 복원] 버튼 클릭
3. JSON 파일 선택 → 확인 → 복원 완료
⚠️ 복원하면 현재 데이터가 덮어씌워집니다
3 다른 기기로 옮기기
1. 현재 기기에서 [📥 데이터 백업] → JSON 다운로드
2. JSON 파일을 새 기기로 전송 (카톡, 드라이브, 메일 등)
3. 새 기기에서 이 HTML 파일을 브라우저로 열기
4. [📤 데이터 복원] → JSON 파일 선택 → 완료
! 데이터가 삭제되는 경우
❌ 브라우저 설정 → '인터넷 사용 기록 삭제' → 쿠키/사이트 데이터 삭제
❌ 삼성 인터넷 → 설정 → 개인정보 → 인터넷 사용 기록 삭제
❌ 크롬 → 설정 → 개인정보 → 인터넷 사용 기록 삭제
❌ 폰 초기화 (공장초기화)
❌ 앱 데이터 삭제 / 앱 삭제 후 재설치
❌ 시크릿 모드(프라이빗 모드)에서 사용 → 탭 닫으면 삭제
💡 위 상황이 발생하기 전에 반드시 백업하세요!
💾 데이터는 이 기기 브라우저에 저장됩니다. 다른 기기로 옮기려면 백업 → 복원을 이용하세요.
