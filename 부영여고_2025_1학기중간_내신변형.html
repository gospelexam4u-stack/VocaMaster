STEP 1 · 첫 조건의 의미
$\sqrt{-a} \cdot \sqrt{b} = -\sqrt{-ab}$ ⟺ $-a < 0$이고 $b < 0$

$\dfrac{\sqrt{2a}}{\sqrt{3c}} = \sqrt{\dfrac{2a}{3c}}$ ⟺ $2a \geq 0$이고 $3c > 0$
$\therefore a > 0, \, c > 0$
STEP 3 · 부호 종합
$a > 0, \, b < 0, \, c > 0$
STEP 4 · 절댓값 풀기
$|a+c| = a+c$ (양수)
$|b-a| = a-b$ ($b-a<0$)
$|c-b| = c-b$ ($c-b>0$)
STEP 5 · 식 정리
$(a+c) + (a-b) - (c-b) = 2a$

$a=5$: $b > 6, b < 10, b \leq 7$ → $b = 7$ (1개)

$1 + 3 + 3 + 3 + 1 = 11$

객관식 ●2-2. 이차함수와 직선

STEP 1 · 교점의 $x$좌표
$x^2 = x + k \Rightarrow x^2 - x - k = 0$
두 근 $\alpha (>0), \, \beta (<0)$에 대해 근과 계수의 관계:
$\alpha + \beta = 1, \quad \alpha\beta = -k$
STEP 2 · 두 삼각형의 넓이
$\triangle AOC$: 밑변 $OC = \alpha$, 높이 $\alpha^2$ → $S_1 = \dfrac{\alpha^3}{2}$
$\triangle DOB$: 밑변 $OD = -\beta$, 높이 $\beta^2$ → $S_2 = -\dfrac{\beta^3}{2}$
STEP 3 · 넓이의 차
$S_1 - S_2 = \dfrac{\alpha^3 + \beta^3}{2} = 17$
$\alpha^3 + \beta^3 = 34$
STEP 4 · 곱셈공식의 변형
$\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta)$
$= 1 - 3(-k)(1) = 1 + 3k$
STEP 5 · $k$ 결정
$1 + 3k = 34 \Rightarrow k = 11$
그림 + 근과 계수의 관계 + 곱셈공식 변형이 결합된 종합 문제. $\alpha^3+\beta^3$ 공식은 반드시 외우기.

서술형 ●2-2. 근의 변환
정답 $x^2 - 7x + 11 = 0$
STEP 1 · 원래 두 근의 합·곱
근과 계수의 관계: $\alpha + \beta = 3, \, \alpha\beta = 1$
STEP 2 · 새 두 근의 합·곱
합: $(\alpha+2) + (\beta+2) = (\alpha+\beta) + 4 = 3 + 4 = 7$
곱: $(\alpha+2)(\beta+2) = \alpha\beta + 2(\alpha+\beta) + 4 = 1 + 6 + 4 = 11$
STEP 3 · 새 이차방정식 작성
$x^2$의 계수가 1인 이차방정식:
$x^2 - (\text{합})x + (\text{곱}) = 0$
$\therefore x^2 - 7x + 11 = 0$
두 근이 $\alpha', \beta'$인 이차방정식 = $x^2 - (\alpha'+\beta')x + \alpha'\beta' = 0$. 합·곱만 알면 끝.

① 다항식의 덧셈·뺄셈

같은 차수끼리 계수를 더한다.
$(2x^2 - x + 3) - (x^2 + 3x - 5)$

$= x^2 - 4x + 8$
⚠ 괄호 앞 $-$ → 안의 모든 부호 반전. $-(-5) = +5$.

② 다항식의 곱셈

한쪽의 모든 항 × 다른 쪽의 모든 항. 빠짐없이.
$(a+b)(c+d) = ac + ad + bc + bd$
특정 차수의 계수만 묻는 문제 → 전체 전개 금지. 그 차수가 되는 짝만 골라 더한다.
$(x^2+3x-1)(2x^2-x+4)$의 $x^3$ 계수
$= x^2 \cdot (-x) + 3x \cdot 2x^2 = -1 + 6 = 5$
↓ 매번 분배법칙으로 풀면 시간 낭비. 자주 나오는 곱셈 패턴은 외워둔다 ↓

③ 곱셈공식 — 외울 것

$(a \pm b)^2 = a^2 \pm 2ab + b^2$
$(a+b)(a-b) = a^2 - b^2$
$(a \pm b)^3 = a^3 \pm 3a^2 b + 3ab^2 \pm b^3$
$(a+b)(a^2 - ab + b^2) = a^3 + b^3$
$(a-b)(a^2 + ab + b^2) = a^3 - b^3$
$(a+b+c)^2 = a^2+b^2+c^2 + 2(ab+bc+ca)$
이 6개는 무조건 암기. 다음 ④부터는 모두 이 공식들을 거꾸로 응용한 것.

④ 곱셈공식의 변형 ★ 핵심

이 단원의 보스 섹션. 1학기 시험에서 가장 자주 등장하는 무기.
$a+b$와 $ab$만 알아도 다음을 즉시 얻는다.
$a^2 + b^2 = (a+b)^2 - 2ab$  필수
$a^3 + b^3 = (a+b)^3 - 3ab(a+b)$  필수
$a^3 - b^3 = (a-b)^3 + 3ab(a-b)$
$a^4 + b^4 = (a^2+b^2)^2 - 2(ab)^2$
위 2개가 핵심. 아래 2개는 위에서 즉시 유도되니 외우지 말고 만드는 법만 알아둘 것.

왜 무기인가? $a, b$의 값을 직접 구할 필요가 없다. 합과 곱만으로 모든 대칭식이 풀린다. 이게 2-2 단원의 시간을 절반으로 줄인다.

⑤ $x$와 $\dfrac{1}{x}$ 형태 ★ 핵심

$x \cdot \dfrac{1}{x} = 1$. 이것 하나가 핵심.
$x^2 + \dfrac{1}{x^2} = \left(x + \dfrac{1}{x}\right)^2 - 2$
$x^2 + \dfrac{1}{x^2} = \left(x - \dfrac{1}{x}\right)^2 + 2$
$x^3 + \dfrac{1}{x^3} = \left(x + \dfrac{1}{x}\right)^3 - 3\left(x + \dfrac{1}{x}\right)$
$x^3 - \dfrac{1}{x^3} = \left(x - \dfrac{1}{x}\right)^3 + 3\left(x - \dfrac{1}{x}\right)$
사실 이건 ④의 변형 공식에 $a = x$, $b = \dfrac{1}{x}$, $ab = 1$을 대입한 것이다. 외울 게 아니라 유도하면 된다.

분모 유리화 예시 · $x = \sqrt{5} + 2$일 때
$\dfrac{1}{x} = \dfrac{1}{\sqrt{5}+2} = \dfrac{\sqrt{5}-2}{(\sqrt{5}+2)(\sqrt{5}-2)} = \dfrac{\sqrt{5}-2}{5-4} = \sqrt{5}-2$

$x + \dfrac{1}{x} = (\sqrt{5}+2) + (\sqrt{5}-2) = 2\sqrt{5}$
$x - \dfrac{1}{x} = (\sqrt{5}+2) - (\sqrt{5}-2) = 4$
$x \cdot \dfrac{1}{x} = (\sqrt{5}+2)(\sqrt{5}-2) = 1$ ✓
분모와 분자에 켤레($\sqrt{5}-2$)를 곱하면 분모가 $(a+b)(a-b) = a^2 - b^2$ 공식으로 정수가 된다.

1-2 항등식 ← 곱셈공식이 곧 항등식. $(a+b)^2 = a^2+2ab+b^2$은 모든 $a, b$에 대해 성립하는 항등식이다.
$(x+a)(x+b) = x^2 + 5x + 6$이 항등식이라면
계수비교: $a + b = 5$, $ab = 6$ → $a=2, b=3$ (또는 $a=3, b=2$)
이 계수비교법이 1-2의 핵심 도구. 1-1의 곱셈공식을 알아야 1-2가 풀린다.

1-3 인수분해 ← 곱셈의 역방향. 곱셈공식을 거꾸로 읽으면 인수분해 공식.
[곱셈]  $(x+2)(x+3) = x^2 + 5x + 6$
[인수분해]  $x^2 + 5x + 6 = (x+2)(x+3)$

[곱셈]  $(a+b)(a^2-ab+b^2) = a^3 + b^3$
[인수분해]  $a^3 + b^3 = (a+b)(a^2-ab+b^2)$
즉 1-1을 외운 공식이 1-3에서 그대로 무기가 된다.

2-2 근과 계수의 관계 ← 합·곱만 알 때 다른 대칭식을 구하는 도구.
이차방정식 $x^2 - 5x + 3 = 0$의 두 근 $\alpha, \beta$
근과 계수의 관계 → $\alpha + \beta = 5$, $\alpha\beta = 3$

$\alpha^2 + \beta^2$를 묻는다면? 1-1의 변형 공식으로 즉시:
$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = 25 - 6 = 19$

$\alpha^3 + \beta^3$을 묻는다면?
$\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta) = 125 - 45 = 80$
$\alpha, \beta$의 값을 직접 구할 필요가 없다. ④의 변형 공식이 2-2의 풀이 시간을 절반으로 줄인다.

① 항등식

모든 $x$에 대해 성립하는 식.
$(x+1)^2 = x^2 + 2x + 1$ ← 어떤 $x$를 넣어도 성립
$2x + 3 = 7$ ← $x=2$에서만 성립 (이건 방정식)
항등식이라는 조건이 주어지면 두 가지 도구를 쓴다.

도구 1 · 계수비교법
$ax^2 + bx + c = 2x^2 - 3x + 5$가 항등식
↓ 같은 차수 계수가 모두 같아야 함
$a = 2, \quad b = -3, \quad c = 5$
도구 2 · 수치대입법
$(a+b)x + (a-b) = 5x + 1$이 항등식
↓ 편한 $x$값을 골라 대입
$x=1$: $2a = 6 \Rightarrow a = 3$
$x=0$: $a-b = 1 \Rightarrow b = 2$
두 방법 중 빠른 쪽을 선택. 차수가 낮으면 계수비교, 차수가 높거나 미지수가 많으면 수치대입(특히 $x=0, 1, -1$이 편함).

② 다항식의 나눗셈

$A = BQ + R$  ($A$: 피제수, $B$: 제수, $Q$: 몫, $R$: 나머지)
$x^3 - 2x^2 + 5x - 4 = (x+1) \cdot Q(x) + R$
↓ 직접 나눗셈 또는 조립제법
$Q(x) = x^2 - 3x + 8, \quad R = -12$
⚠ 나머지의 차수는 항상 제수의 차수보다 작다.
• $x - c$로 나누면 나머지는 상수 (차수 0)
• $x^2 + \cdots$로 나누면 나머지는 일차식 또는 상수 ($ax+b$ 형태)
• $x^3 + \cdots$로 나누면 나머지는 이차식 이하 ($ax^2+bx+c$ 형태)
↓ 나눗셈을 직접 안 해도 나머지를 알 수 있는 방법 ↓

③ 나머지정리 ★ 핵심

이 단원의 보스 섹션. 6개 원본 중 4개가 이걸로 풀린다.
다항식 $f(x)$를 $x - c$로 나눈 나머지 $= f(c)$  필수
나눗셈을 직접 할 필요가 없다. $x = c$를 대입한 값이 곧 나머지.
$f(x) = x^3 - 2x^2 + 5x - 4$를 $x + 1$로 나눈 나머지
$= f(-1) = -1 - 2 - 5 - 4 = -12$
일반화
$f(x)$를 $ax - b$로 나눈 나머지 $= f\left(\dfrac{b}{a}\right)$
즉 제수가 $0$이 되는 $x$값을 $f$에 대입하면 끝.

사용 패턴
• "나누어떨어진다" → $f(c) = 0$
• "나머지가 $r$" → $f(c) = r$
• 두 조건 동시 → $f(c_1), f(c_2)$ 값을 연립방정식으로

④ 인수정리

나머지정리의 특수한 경우. 나머지가 $0$이면 인수.
$f(c) = 0 \iff (x - c)$가 $f(x)$의 인수  필수
$f(x) = x^3 - 2x^2 - 5x + 6$
$f(1) = 1 - 2 - 5 + 6 = 0$ → $x - 1$이 인수
↓ 조립제법으로 나누면
$f(x) = (x - 1)(x^2 - x - 6) = (x-1)(x-3)(x+2)$
고차 다항식 인수분해의 첫 단계는 항상 인수정리. $f(\pm 1), f(\pm 2), f(\pm 3)$을 차례로 시도해서 $0$이 되는 값을 찾는다.

⑤ 조립제법

$x - c$로 나눌 때 쓰는 빠른 계산법. 직접 나눗셈보다 빠르다.
첫 계수 내림 → $c$ 곱해 다음 자리에 더함 → 마지막 칸이 나머지  필수
$x^3 - 2x^2 + 5x - 4$를 $x + 1$로 나눌 때 ($c = -1$)

$\begin{array}{c|rrrr} -1 & 1 & -2 & 5 & -4 \\ & & -1 & 3 & -8 \\ \hline & 1 & -3 & 8 & \boxed{-12} \end{array}$

몫: $x^2 - 3x + 8$,    나머지: $-12$
⚠ $x + 1$로 나누면 $c = -1$. 부호 실수 잦음.
⚠ 빠진 차수는 $0$으로 채워서 시작. 예: $x^3 + 2x - 1$은 $x^2$ 자리에 $0$.

⑥ 큰 제수의 나머지 → 작은 제수로

$f(x)$를 $x^3 + 1$로 나눈 나머지를 묻는데, 이미 $f(x)$를 $x^2 - x + 1$로 나눈 나머지가 주어졌다면?
$x^3 + 1 = (x+1)(x^2 - x + 1)$ ← 인수분해
↓
"$f(x)$를 $x^3+1$로 나눈 나머지"를 $x^2-x+1$로 다시 나눈 결과 = 원래 주어진 나머지
핵심 트릭 · $x^2 - x + 1 = 0$일 때 $x^2 = x - 1$. 이 관계로 $ax^2 + b$ 같은 식에서 $x^2$ 자리를 $x-1$로 치환해 차수를 낮춘다.
$ax^2 + b = a(x - 1) + b = ax + (b - a)$
이 식이 주어진 나머지 $2x + 3$과 같다면
$a = 2, \quad b - a = 3 \Rightarrow b = 5$
제수가 일차식 곱일 때는 더 직접적이다. 제수 $(x-1)(x+2)(x-3)$이면 나머지는 $ax^2+bx+c$ 형태. 세 점 $f(1), f(-2), f(3)$을 연립방정식으로 푼다.

⑦ 응용 트릭 3가지 ★ 핵심

치환 나머지정리, $f(x)-k$ 인수분해, 연쇄 나머지 — 이 단원의 응용 중 가장 자주 등장하는 3가지.

트릭 1 · 치환 나머지정리
$f(g(x))$를 $x - c$로 나눈 나머지 $= f(g(c))$
$f(x)$가 주어졌을 때 $f(x^2 + 1)$을 $x - 2$로 나눈 나머지
$= f(2^2 + 1) = f(5)$
대입을 두 번 거친다. $x = 2$를 $x^2+1$에 먼저 → $5$. 그 다음 $f(5)$.

트릭 2 · "여러 점에서 같은 값" → $f(x) - k$ 인수분해
$f(c_1) = f(c_2) = f(c_3) = k$이고 $f$가 삼차식이면
$f(x) - k = a(x - c_1)(x - c_2)(x - c_3)$
$f(1) = f(2) = f(4) = 2$, $f(0) = 10$인 삼차식
↓
$f(x) - 2 = a(x-1)(x-2)(x-4)$
$f(0) = 10$ 대입: $-2 + a \cdot (-1)(-2)(-4) = 10 - 2$
$-8a = 8 \Rightarrow a = -1$
$f(x) = -(x-1)(x-2)(x-4) + 2$
트릭 3 · 연쇄 나머지
한 나머지 조건으로 미지 계수 결정 → 확정된 다항식에 다시 나머지정리 적용. 나머지정리를 두 번 사용.
$f(x) = x^3 - 2x^2 - 5x + a$를 $x-3$으로 나눈 나머지가 $7$
↓ 1차: $f(3) = 7$로 $a$ 결정
$27 - 18 - 15 + a = 7 \Rightarrow a = 13$
↓ 2차: $f(-1)$ 계산
$f(-1) = -1 - 2 + 5 + 13 = 15$

⑧ 관련 단원

$x^3 - 6x^2 + 11x - 6$의 인수분해

$= (x-1)(x^2 - 5x + 6) = (x-1)(x-2)(x-3)$

① 기본 인수분해 공식

곱셈공식을 거꾸로 읽으면 곧 인수분해 공식.
$a^2 - b^2 = (a+b)(a-b)$  필수
$a^2 \pm 2ab + b^2 = (a \pm b)^2$  필수
$a^3 + b^3 = (a+b)(a^2 - ab + b^2)$  필수
$a^3 - b^3 = (a-b)(a^2 + ab + b^2)$  필수
$a^3 \pm 3a^2b + 3ab^2 \pm b^3 = (a \pm b)^3$
$a^2 + b^2 + c^2 + 2(ab+bc+ca) = (a+b+c)^2$
이게 1-1에서 외운 곱셈공식과 동일하다. 외운 공식은 두 번 쓰인다 — 곱셈에서 한 번, 인수분해에서 한 번.

$\Rightarrow (x+2)(x+3)$

② 치환 인수분해 ★ 핵심

이 단원의 보스 섹션. 4차 이상의 식을 인수분해할 때 가장 강력한 도구.
공통 부분을 한 문자 $t$로 치환해서 이차식 문제로 바꾼다.
공통 부분 발견 → $t$로 치환 → 이차식 인수분해 → $t$ 복원 → 추가 분해 검토
예시 1 · 두 이차식 곱 + 상수
$(x^2-4x+3)(x^2-4x+5) - 3$
↓ 공통 부분 $x^2 - 4x = t$로 치환
$(t+3)(t+5) - 3 = t^2 + 8t + 12 = (t+2)(t+6)$
↓ $t$ 복원
$(x^2 - 4x + 2)(x^2 - 4x + 6)$
예시 2 · $x(x \pm 1)$ 형태
$x^2(x+1)^2 + 2x^2 + 2x + k$
↓ $x(x+1) = x^2 + x = t$로 치환
$t^2 + 2t + k$
↓ 완전제곱이 되려면 $k = 1$
$(t+1)^2 = (x^2 + x + 1)^2$
⚠ 치환 후 끝내지 말 것. $t$를 복원한 뒤 이차식이 또 인수분해되는지 반드시 확인. $(x^2 - 4x + 2)$ 같은 식이 더 분해될 수 있는지 판별식으로 검토.

완전제곱 조건 · 판별식 $= 0$
$t^2 + bt + c$가 완전제곱식 $\iff$ $b^2 - 4c = 0$
즉 $c = \dfrac{b^2}{4}$

③ 공통인수 묶기 · 그룹화

공통인수 묶기 · 모든 항에 들어있는 것을 빼낸다.
$3x^3 + 6x^2 - 9x = 3x(x^2 + 2x - 3) = 3x(x+3)(x-1)$
그룹화 인수분해 · 4개 이상의 항을 둘씩 묶어 공통인수를 만든다.
$a^3 + ab^2 - a^2 b - b^3$
↓ $a$와 $b$로 묶기
$= a(a^2 + b^2) - b(a^2 + b^2)$
$= (a - b)(a^2 + b^2)$
$a^2 - b^2 - ac + bc$
↓ $a^2 - b^2$과 $-ac + bc$로 묶기
$= (a+b)(a-b) - c(a-b)$
$= (a - b)(a + b - c)$
공통 부분이 보일 때까지 식을 재배열하는 게 핵심. 한 번에 안 보이면 항의 순서를 바꿔본다.

④ 인수정리 활용 (고차 다항식)

3차 이상의 다항식은 1-2의 인수정리로 시작한다.
$f(c) = 0$인 $c$를 찾으면 $(x - c)$가 인수
$c$를 시도하는 순서: $\pm 1, \pm 2, \pm 3, \ldots$ (상수항의 약수 중에서)
$f(x) = x^3 - 6x^2 + 11x - 6$
$f(1) = 1 - 6 + 11 - 6 = 0$ → $(x-1)$ 인수
↓ 조립제법으로 나누면
$f(x) = (x-1)(x^2 - 5x + 6) = (x-1)(x-2)(x-3)$
표준 절차: 인수정리로 한 인수 찾기 → 조립제법으로 차수 낮추기 → 남은 이차식 인수분해. 1-2의 ④, ⑤가 1-3에서 그대로 무기.

⑤ 삼각형 모양 결정 (활용)

세 변 $a, b, c$의 관계식이 주어지고 삼각형 모양을 결정하는 문제. 인수분해 + 삼각형 부등식이 핵심.
조건: $a^3 + ab^2 - a^2 b - b^3 = 0$
↓ 인수분해
$(a - b)(a^2 + b^2) = 0$
↓ $a, b > 0$이므로 $a^2 + b^2 > 0$
$\therefore a = b$ → 이등변삼각형
삼각형 부등식 · 모든 삼각형은 다음을 만족.
$a + b > c, \quad b + c > a, \quad c + a > b$
활용 패턴 · 인수분해 결과 중 $(a + b - c)$ 같은 인수가 나오면, 삼각형 부등식에 의해 이 값은 $> 0$이므로 $0$이 될 수 없다. 다른 인수가 $0$이어야 한다.
$a^2 - b^2 - ac + bc = (a - b)(a + b - c) = 0$
$a + b - c \neq 0$이므로 $a = b$

① 복소수의 정의

$i^2 = -1$로 정의되는 새로운 수 $i$를 도입하면, 모든 수는 $a + bi$ 꼴로 표현된다.

$a$ = 실수부 (real part),   $b$ = 허수부 (imaginary part)
$z = 3 + 2i$ → 실수부 $3$, 허수부 $2$  (허수부는 $2i$가 아니라 $2$)
$z = -5$ → 실수부 $-5$, 허수부 $0$ (실수)
$z = 4i$ → 실수부 $0$, 허수부 $4$ (순허수)
두 복소수가 같다 $\iff$ 실수부와 허수부가 각각 같다.
$a + bi = c + di \iff a = c \text{ 그리고 } b = d$

② 복소수의 사칙연산

덧셈·뺄셈 · 실수부끼리, 허수부끼리
$(2 + 3i) + (4 - i) = 6 + 2i$

곱셈 · 분배법칙 + $i^2 = -1$
$(2 + 3i)(1 + i) = 2 + 2i + 3i + 3i^2 = 2 + 5i - 3 = -1 + 5i$

③ 켤레복소수와 분모 유리화

켤레복소수 · 허수부의 부호만 바꾼 것.
$z = a + bi$의 켤레 $\bar{z} = a - bi$
켤레의 핵심 성질:
$z \cdot \bar{z} = (a+bi)(a-bi) = a^2 - (bi)^2 = a^2 + b^2$  (실수)
이게 분모 유리화의 원리. 분수의 분모에 켤레를 곱하면 분모가 실수가 된다. 예시 · $\dfrac{1}{1+i}$ 유리화
$\dfrac{1}{1+i} \cdot \dfrac{1-i}{1-i} = \dfrac{1-i}{(1+i)(1-i)} = \dfrac{1-i}{1+1} = \dfrac{1-i}{2} = \dfrac{1}{2} - \dfrac{1}{2}i$
예시 · $\dfrac{3i}{1+i}$ 유리화
$\dfrac{3i}{1+i} \cdot \dfrac{1-i}{1-i} = \dfrac{3i(1-i)}{2} = \dfrac{3i - 3i^2}{2} = \dfrac{3 + 3i}{2}$
모든 복소수 답은 $a + bi$ 꼴로 정리되어야 한다. 분모에 $i$가 있으면 미완성.

④ $i$의 주기와 핵심값 ★ 핵심

이 단원의 보스 섹션. 12문제 중 8문제가 이걸로 풀린다.
$i^1 = i, \quad i^2 = -1, \quad i^3 = -i, \quad i^4 = 1$  필수
↓ 이후 주기 $4$로 무한 반복
$i^n$의 값은 $n$을 $4$로 나눈 나머지로 결정
$i^{17} = i^{4 \cdot 4 + 1} = i^1 = i$
$i^{50} = i^{4 \cdot 12 + 2} = i^2 = -1$
$i^{99} = i^{4 \cdot 24 + 3} = i^3 = -i$
$i^{100} = i^{4 \cdot 25} = i^4 = 1$
반드시 외울 핵심값 4개
$\dfrac{1+i}{1-i} = i$  필수
$\dfrac{1-i}{1+i} = -i$  필수

⑤ 음수의 제곱근 부호 규칙 ★ 핵심

$\sqrt{a}\sqrt{b}$가 항상 $\sqrt{ab}$인 것은 아니다. $a, b$의 부호에 따라 부호가 바뀐다.
$\sqrt{a}\sqrt{b} = -\sqrt{ab}$  $\iff$  $a < 0$ 그리고 $b < 0$  필수
$\dfrac{\sqrt{a}}{\sqrt{b}} = -\sqrt{\dfrac{a}{b}}$  $\iff$  $a > 0$ 그리고 $b < 0$  필수
이 두 규칙만 기억하면 모든 부호 결정 문제가 해결된다.

왜 그럴까?
$\sqrt{-2}\sqrt{-3} = (\sqrt{2}i)(\sqrt{3}i) = \sqrt{6} \cdot i^2 = -\sqrt{6}$
$\sqrt{(-2)(-3)} = \sqrt{6}$
$\therefore \sqrt{-2}\sqrt{-3} = -\sqrt{6} = -\sqrt{(-2)(-3)}$
음수의 제곱근은 $i$를 포함하므로, $i \cdot i = -1$이 곱해지면서 부호가 바뀐다.

활용 패턴 · 조건이 주어지면 $a, b$의 부호를 먼저 결정.
$\sqrt{a}\sqrt{b} = -\sqrt{ab}$이고 $|a| > |b|$일 때
↓ 부호 규칙: $a < 0, b < 0$
↓ $|a| > |b|$ 그리고 둘 다 음수면 $a < b < 0$
↓ 이제 $|a+b|, |a-b|$ 등의 부호를 결정하고 절댓값 풀기

② 판별식 · 근의 종류

판별식 $D = b^2 - 4ac$는 근의 공식의 제곱근 안의 식. 부호로 근의 종류를 결정.
$D > 0$  →  서로 다른 두 실근  필수
$D = 0$  →  중근 (실근 1개)  필수
$D < 0$  →  서로 다른 두 허근  필수
활용 패턴
• "실근을 가질 조건" → $D \geq 0$
• "중근을 가질 조건" → $D = 0$
• "허근을 가질 조건" → $D < 0$
• "$x$축과 한 점에서 만남" → $D = 0$ (이차함수가 $x$축 접함)
• "$x$축과 만나지 않음" → $D < 0$

모든 $k$에 대해 중근 같은 조건은 $k$에 대한 항등식으로 처리.
$x^2 - 2(k+a)x + k^2 + bk + 4 = 0$이 모든 $k$에 대해 중근
↓ $D = 0$ 식을 세우고 $k$로 정리
$k$의 항등식으로 보고 1-2의 계수비교법 적용

③ 근과 계수의 관계 ★ 핵심

이 단원의 보스 섹션. 7개 원본 중 5개가 이걸로 풀린다.
$ax^2 + bx + c = 0$의 두 근 $\alpha, \beta$
$\alpha + \beta = -\dfrac{b}{a}$  필수
$\alpha \beta = \dfrac{c}{a}$  필수
$\alpha, \beta$의 값을 직접 구하지 않아도 합과 곱은 즉시 알 수 있다.
$x^2 - 5x + 3 = 0$의 두 근 $\alpha, \beta$
$\alpha + \beta = 5, \quad \alpha\beta = 3$
1-1의 변형 공식과 결합 → 이 단원의 진짜 무기
$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta$
$\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta)$
$(\alpha-\beta)^2 = (\alpha+\beta)^2 - 4\alpha\beta$
$\dfrac{1}{\alpha} + \dfrac{1}{\beta} = \dfrac{\alpha+\beta}{\alpha\beta}$
$\dfrac{\alpha}{\beta} + \dfrac{\beta}{\alpha} = \dfrac{\alpha^2+\beta^2}{\alpha\beta}$
위의 $x^2 - 5x + 3 = 0$에서
$\alpha^2 + \beta^2 = 25 - 6 = 19$
$\alpha^3 + \beta^3 = 125 - 45 = 80$
$(\alpha-\beta)^2 = 25 - 12 = 13$
근의 공식으로 $\alpha, \beta$를 직접 구하면 시간 5배. 이 변형 공식이 그 시간을 절약한다.

두 근이 $p, q$이고 $x^2$의 계수가 $1$인 이차방정식
$\Rightarrow x^2 - (p + q)x + pq = 0$  필수
새 근의 합과 곱만 알면 끝.

대표 패턴 5가지
원래 근 $\alpha, \beta$에서 새 근 $p, q$가 다음과 같을 때:

① $p = \alpha + 1, q = \beta + 1$ (1씩 더함)
새 합 $= \alpha + \beta + 2$,   새 곱 $= (\alpha+1)(\beta+1) = \alpha\beta + (\alpha+\beta) + 1$

② $p = 2\alpha, q = 2\beta$ ($k$배)
새 합 $= k(\alpha+\beta)$,   새 곱 $= k^2 \alpha\beta$  (주의: 곱은 $k^2$배)

③ $p = \alpha^2, q = \beta^2$ (제곱)
새 합 $= \alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta$  (③의 변형 공식)
새 곱 $= \alpha^2 \beta^2 = (\alpha\beta)^2$

④ $p = \dfrac{\alpha}{\beta}, q = \dfrac{\beta}{\alpha}$ (비)
새 합 $= \dfrac{\alpha^2+\beta^2}{\alpha\beta}$,   새 곱 $= 1$  (곱은 항상 1)
$f(2x-1) = 0$ 같은 치환형도 같은 원리.
$f(x) = 0$의 두 근이 $\alpha, \beta$일 때
$f(2x-1) = 0$의 두 근은 $2x - 1 = \alpha$, $2x - 1 = \beta$를 만족하는 $x$
$\Rightarrow x = \dfrac{\alpha+1}{2}, \dfrac{\beta+1}{2}$

⑤ 이차함수의 그래프와 표준형

표준형 · 꼭짓점이 한눈에 보임.
$y = a(x - p)^2 + q$
꼭짓점 $(p, q)$,   축 $x = p$
일반형 → 표준형 · 완전제곱으로 변형.
$y = x^2 - 4x + 7$
$\;\;\, = (x^2 - 4x + 4) - 4 + 7$
$\;\;\, = (x - 2)^2 + 3$
↓ 꼭짓점 $(2, 3)$, 축 $x = 2$
위/아래 볼록
• $a > 0$ → 아래로 볼록, 꼭짓점에서 최솟값
• $a < 0$ → 위로 볼록, 꼭짓점에서 최댓값

두 근으로부터 이차함수 표현
$x$축과 만나는 점이 $\alpha, \beta$인 이차함수
$\Rightarrow y = a(x - \alpha)(x - \beta)$
이때 꼭짓점 $x$좌표 $= \dfrac{\alpha + \beta}{2}$ (두 근의 중점)

⑥ 제한 구간의 최대·최솟값 ★ 핵심

꼭짓점이 구간 안인지 밖인지가 모든 것을 결정.
원본 16번이 이 패턴.
구간 $[m, n]$에서 이차함수 $y = a(x-p)^2 + q$의 최대·최솟값
↓ 꼭짓점 $x = p$가 구간 안($m \leq p \leq n$)인지 밖인지로 분기
경우 1 · 꼭짓점이 구간 안
$a > 0$이면: 최솟값 = $f(p) = q$,   최댓값 = $f(m), f(n)$ 중 큰 것
$a < 0$이면: 최댓값 = $f(p) = q$,   최솟값 = $f(m), f(n)$ 중 작은 것
경우 2 · 꼭짓점이 구간 밖
이차함수가 구간 내에서 단조함수 (계속 증가 또는 계속 감소)
양 끝점 $f(m), f(n)$ 중 한쪽이 최댓값, 다른 쪽이 최솟값
⚠ $a$의 부호를 모르는 경우 → $a > 0$과 $a < 0$ 두 경우 모두 검토.
원본 16번이 모두 이 형태. 한 경우만 답하면 절반 감점.

실전 예시
$f(x) = a(x-2)(x-4)$, 구간 $[0, 1]$에서 최솟값 $12$
↓ 꼭짓점 $x = 3$은 구간 $[0,1]$의 오른쪽 밖
↓ 구간 내에서 단조함수

$a > 0$이면 꼭짓점 멀어질수록 큼 → $x = 1$에서 최솟값
$f(1) = a \cdot (-1)(-3) = 3a = 12 \Rightarrow a = 4$ ✓

$a < 0$이면 꼭짓점 멀어질수록 작음 → $x = 0$에서 최솟값
$f(0) = a \cdot (-2)(-4) = 8a = 12 \Rightarrow a = \dfrac{3}{2}$, $a < 0$과 모순 ✗

⑦ 이차함수와 직선의 교점

두 식을 같다고 두면 이차방정식이 나오고, 그 방정식의 근이 곧 교점의 $x$좌표.
$y = x^2$과 $y = 2x + k$의 교점
↓ 같다고 두면
$x^2 = 2x + k \Rightarrow x^2 - 2x - k = 0$
↓ 두 근 $\alpha, \beta$ → 근과 계수의 관계 적용 가능
핵심 통찰 · 교점의 $x$좌표가 곧 이차방정식의 근. ③의 근과 계수의 관계 + 1-1의 변형 공식이 그대로 적용된다.

활용 패턴
교점 두 점 $A(\alpha, \alpha^2)$, $B(\beta, \beta^2)$ 사이 거리, 삼각형 넓이, 사다리꼴 넓이
↓ 모두 $\alpha + \beta$, $\alpha\beta$로 표현 가능
$|AB|^2 = (\beta - \alpha)^2 + (\beta^2 - \alpha^2)^2$
$\;\;\;\;\,= (\beta - \alpha)^2 [1 + (\alpha + \beta)^2]$
구간함수와 직선 (원본 23번 패턴)
함수가 두 부분으로 나뉜 구간함수 $f(x) = \begin{cases} x^2 - 9 & |x| > 3 \\ -x^2 + 9 & |x| \leq 3 \end{cases}$의 경우, 직선 $y = mx + k$와의 교점은 각 부분에서 따로 계산.
중앙 부분 $-x^2+9 = mx+k$ → 이차방정식, 근의 개수
외부 부분 $x^2-9 = mx+k$ → 이차방정식, 근의 개수
↓ 단, 외부 근이 정말 외부($|x| > 3$)에 있는지 확인 필요
↓ 중앙 근도 정말 중앙($|x| \leq 3$)에 있는지 확인 필요
↓ 두 부분의 근 개수 합 = 총 교점 수
⚠ 경계점($x = \pm 3$)에서는 두 식이 같은 값($0$)을 가지므로, 한 점이 양쪽에 모두 카운트되지 않게 주의.

1-1 곱셈공식의 변형 ← 이 단원의 진짜 무기. ③ 근과 계수의 관계로 $\alpha+\beta$, $\alpha\beta$를 얻은 뒤 변형 공식으로 $\alpha^2+\beta^2$, $\alpha^3+\beta^3$ 등을 즉시 계산.
$x^2 - 5x + 3 = 0$의 두 근 $\alpha, \beta$
$\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta) = 125 - 45 = 80$

↓ 인수분해
$(x-2)(x-3) = 0 \Rightarrow x = 2 \text{ 또는 } 3$
2-1 복소수 ← ② 판별식 $D < 0$이면 허근. 실수 계수 이차방정식의 두 허근은 항상 켤레쌍.
$x^2 + 2x + 5 = 0$
$\Rightarrow x = -1 \pm 2i$  (켤레쌍)

정답 & 모범 풀이
정답 몫 $x^2-3x+8$, 나머지 $-12$
1STEP 1 · $f(1) = 0$ 조건
$x-1$로 나누어떨어지므로:
$f(1) = 1 + a + b - 4 = a + b - 3 = 0$
$\Rightarrow a + b = 3$
2STEP 2 · $f(2) = 6$ 조건
$f(2) = 8 + 4a + 2b - 4 = 4a + 2b + 4 = 6$
$\Rightarrow 4a + 2b = 2 \;\Rightarrow\; 2a + b = 1$
3STEP 3 · 연립 풀이
두 식을 빼서 $a$ 결정:
$(2a+b) - (a+b) = 1 - 3$
$a = -2,\quad b = 3 - a = 5$

$f(x) = x^3 - 2x^2 + 5x - 4$를 $x+1$로 나눕니다. $d = -1$.
$\begin{array}{c|cccc}-1 & 1 & -2 & 5 & -4 \\ & & -1 & 3 & -8 \\ \hline & 1 & -3 & 8 & -12\end{array}$
5STEP 5 · 결론
몫 $= x^2 - 3x + 8$, 나머지 $= -12$

정답 & 모범 풀이
정답 $x^2 - 2x + 3 = 0$
1STEP 1 · 원래 근의 합·곱
근과 계수의 관계:
$\alpha + \beta = \dfrac{4}{2} = 2$
$\alpha\beta = \dfrac{3}{2}$
2STEP 2 · 새 근의 합
$(2\alpha - 1) + (2\beta - 1) = 2(\alpha+\beta) - 2$
$= 2 \cdot 2 - 2 = 2$
3STEP 3 · 새 근의 곱
$(2\alpha-1)(2\beta-1) = 4\alpha\beta - 2(\alpha+\beta) + 1$
$= 4 \cdot \dfrac{3}{2} - 2 \cdot 2 + 1 = 6 - 4 + 1 = 3$
4STEP 4 · 이차방정식 구성
두 근이 $p, q$이고 $x^2$의 계수가 $1$인 이차방정식은 $x^2 - (p+q)x + pq = 0$.
$x^2 - 2x + 3 = 0$

채점 기준 — ① 중앙 근 범위 분석 (3점) ② 외부 근 범위 분석 (3점) ③ 각 $k$ 구간의 교점 수 표 (2점) ④ 두 점 조건 결론 (2점). 이 문제는 전체 교점 수 함수를 완전히 이해해야 해결.
문제 23-3 2-2. 이차방정식과 이차함수

함수 $f(x)$가 다음과 같이 정의되어 있다.
$$f(x) = \begin{cases} x^2 - 9, & x < -3 \text{ 또는 } x > 3 \\ -x^2 + 9, & -3 \leq x \leq 3 \end{cases}$$이 함수의 그래프와 직선 $y = -2x + k$가 서로 다른 세 점에서 만나도록 하는 모든 실수 $k$의 값의 합을 구하는 과정을 서술하시오.

정답 & 모범 풀이
정답 합 $= 16$
1STEP 1 · 함수의 대칭성 관찰
$f(x)$는 $y$축 대칭함수 ($f(-x) = f(x)$).
2STEP 2 · 중앙 접선 조건
$-x^2+9 = -2x+k$, $x^2-2x+(k-9)=0$. 접선 조건 $D = 0$:
$4 - 4(k-9) = 0 \Rightarrow k = 10$
접점 $x = 1 \in [-3, 3]$ ✓
3STEP 3 · $k = 10$의 외부 교점
$x^2-9 = -2x+10 \Rightarrow x^2 + 2x - 19 = 0$. 근 $x = -1 \pm 2\sqrt{5}$.
$-1+2\sqrt{5} \approx 3.47 > 3$ ✓
$-1-2\sqrt{5} \approx -5.47 < -3$ ✓
외부 2점 + 중앙 접점 1점 = 3점 ✓
4STEP 4 · 경계점 $(3, 0)$ 통과
$0 = -6 + k \Rightarrow k = 6$.
5STEP 5 · $k = 6$ 교점 확인
중앙: $-x^2+9 = -2x+6 \Rightarrow x^2-2x-3 = 0 \Rightarrow (x-3)(x+1) = 0$. $x = 3$ (경계 ✓), $x = -1$ (내부). 2점.
외부: $x^2-9 = -2x+6 \Rightarrow x^2+2x-15 = 0 \Rightarrow (x+5)(x-3) = 0$. $x = -5$ ✓, $x = 3$ (외부 아님). 1점.
합계 2+1 = 3점 ✓
6STEP 6 · 합
$k = 6 + 10 = \mathbf{16}$
채점 기준 — ① 중앙 접선 조건 (3점) ② $k=10$ 3점 확인 (2점) ③ 경계점 통과 조건 (2점) ④ $k=6$ 3점 확인 (2점) ⑤ 합 (1점). 기울기만 $+2 \to -2$로 바뀌어도 함수의 $y$축 대칭성 때문에 답은 동일.
문제 23-4 2-2. 이차방정식과 이차함수

함수 $f(x)$가 다음과 같이 정의되어 있다.
$$f(x) = \begin{cases} x^2 - 9, & x < -3 \text{ 또는 } x > 3 \\ -x^2 + 9, & -3 \leq x \leq 3 \end{cases}$$이 함수의 그래프와 직선 $y = 2x + k$가 서로 다른 네 점에서 만나도록 하는 모든 정수 $k$의 값의 합을 구하는 과정을 서술하시오.

정답 & 모범 풀이
정답 합 $= 24$
1STEP 1 · 중앙 교점 개수 분석
$-x^2+9 = 2x+k$, $x^2+2x+(k-9) = 0$. 근 $-1 \pm \sqrt{10-k}$.
$6 \leq k < 10$: 2개
$-6 \leq k < 6$: 1개
그 외 $k$: 0개 혹은 1개 (경계)
2STEP 2 · 외부 교점 개수 분석
$x^2-9 = 2x+k$, $x^2-2x-(9+k) = 0$. 근 $1 \pm \sqrt{10+k}$.
$k > 6$: 2개
$-6 < k \leq 6$: 1개
$k \leq -6$: 0개
3STEP 3 · 네 점 조건
총 교점이 $4$개 $\Leftrightarrow$ 중앙 2개 + 외부 2개.
중앙 2개: $6 \leq k < 10$
외부 2개: $k > 6$
교집합: $6 < k < 10$
4STEP 4 · 경계 재확인
$k = 6$에서 총 3점 (네 점 아님), $k = 10$에서 총 3점. 따라서 네 점은 $k$가 엄격히 $6 < k < 10$.
5STEP 5 · 정수 $k$ 찾기
$6 < k < 10$인 정수.
$k \in \{7, 8, 9\}$
6STEP 6 · 합 계산
$7 + 8 + 9 = \mathbf{24}$
채점 기준 — ① 중앙·외부 개수 분석 (각 2점) ② 네 점 조건 $6 < k < 10$ (3점) ③ 정수 $k$ 나열 (1점) ④ 합 (1점). "네 점 = 중앙 2 + 외부 2"로 분해하는 것이 핵심 통찰.
